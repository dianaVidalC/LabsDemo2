Prueba 2

La prueba consistía en implementar una pequeña modificación al plugin jQuery-Seat-Charts, el cuál
se utiliza para reservar un asiento, donde se debe agregar el nombre y hacer una pequeña validación.

El código que pude aumentar permite que aparezca un modal al hacer click sobre un asiento disponible
solicitando que ingrese el nombre de quien ocupará el asiento.

Intenté asociar un atributo del elemento seleccionado al nombre, lo cual permitiría la validación
para que no se repita el mismo nombre.
